📖  How to use

CSV Format

  • ▸Key column: CAPRODUCTID
  • ▸Source A columns: A_COLNAME
  • ▸Source B columns: B_COLNAME
  • ▸Changed columns list: DIFF_COLUMNS
  • ▸Encoding: UTF-8, comma-separated

Color Legend

  • ▸A value — source A, changed cell
  • ▸B value — source B, changed cell
  • ▸═ — same value in both sources
  • ▸NULL — empty / missing value

Filtering

  • ▸Search box: filter rows by key or any value
  • ▸Column dropdown: show only rows where a specific column differs
  • ▸Right-click on any column header → value filter popup
  • ▸Active column filters marked with ▼

Other

  • ▸Click column header to sort ascending / descending
  • ▸Only changed cells toggle: hide identical values
  • ▸Page size: 25 / 50 / 100 / 200 rows per page
  • ▸🌙 / ☀️ button: toggle dark / light theme
